Gesuido

Description

Gesuido is a traditional turn-based roguelike game with early Macintosh-like 1-bit color (black/white) pixel art. The levels are randomly generated and you will lose everything when you die (permadeath). The game is set in underground sewer beneath Tokyo city. The player is a secret agent for the Japanese government, sent into the sewers below Tokyo on a special mission to destroy a robotics laboratory that threatens all of humanity.

Gesuidoは、iOS向けのローグライクゲームです。ターン制で、死んだら一からやりなおしで、ゲームレベルは毎回自動生成されるという、元祖Rogueに近いRPGです。初期Macintoshのような白黒アイコンで画面が構成され、登場人物が顔アイコンのみで表現されるのも特徴です。プレイヤーは日本政府の特命を受けた隠密となり、東京の地下の下水道を探索し、人類の脅威とされるロボット研究所の破壊工作をめざします。

Features

  • The game screen is filled with 32 x 32 black and white pixel art icons, just like the early Macintosh Finder.
  • Traditional rouguelike elements: turn-based, tile-based, permadeath and procedural generated levels
  • The UI is designed to play while holding a smartphone vertically in one hand.
  • Cute and stylish illustrations by [[[モニョチタポミチ]]] for the [[[「奥義」]]] you can learn as you play.
  • 初期MacintoshのFinderと同じ、32x32ピクセル白黒2値のアイコンで構成された画面
  • 元祖Rogueに近いローグライクの特徴：ターン制、タイルベース、恒久的な死(permadeath)、自動生成レベル
  • スマートフォン縦持ちで片手でも遊べるように設計されたUI
  • ゲーム中で身につけられる「奥義」は、モニョチタポミチによるポップでかわいいイラストで表現
